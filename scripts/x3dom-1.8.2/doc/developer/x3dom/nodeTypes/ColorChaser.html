.nodeTypes. ColorChaser

The ColorChaser animates transitions for single color values. Whenever the set_destination field receives a floating point number, the value_changed creates a transition from its current value to the newly set number. It creates a smooth transition that ends duration seconds after the last number has been received.

<ColorChaser destination='0,0,0' duration='1' initialDestination='0.8,0.8,0.8' initialValue='0.8,0.8,0.8' isActive='false' metadata='X3DMetadataObject' value='0,0,0' ></ColorChaser>
