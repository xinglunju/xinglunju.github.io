.nodeTypes. PixelTexture3D

The PixelTexture3D node defines a 3D image-based texture map as an explicit array of pixel values (image field) and parameters controlling tiling repetition of the texture onto geometry.

HTML Encoding and Default Values⤴

<PixelTexture3D crossOrigin='""' metadata='X3DMetadataObject' origChannelCount='0' repeatS='true' repeatT='true' scale='true' textureProperties='x3dom.nodeTypes.TextureProperties' url='[]' ></PixelTexture3D>
