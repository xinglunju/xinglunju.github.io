This code can be run with ‘python TDViz.py’, as long as you have all the required python packages installed. A snapshot of the interface is shown below. It uses the same packages as the L1551IRS5 version, so you still need to install some stuff first.


 

Here you can select your own FITS files. After you select one, the starting and ending pixels in three axes, and the maximum and minimum values will be updated respectively depending on properties of your datacube. So the default scene plotted will be the entire datacube. If you want to focus on a smaller region and a narrow velocity range, you can change these values and click the ‘Plot’ button again.




It is certain that many known and unknown bugs exist. If you find one, please do not hesitate to inform me, thanks!
